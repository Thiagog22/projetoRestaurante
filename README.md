# projetoRestaurante

Nosso projeto consiste na criação de um site para os serviços de um restaurante, oferecendo funcionalidades como realização de pedidos, página “Quem Somos”, localização, entre outras. O objetivo é automatizar a gestão de um restaurante fictício, aprimorando tanto a administração quanto os serviços oferecidos.
Na página inicial (Home), apresentaremos brevemente alguns de nossos pratos, para que o cliente tenha uma ideia do que pode pedir ou encontre inspiração. Além disso, haverá uma mensagem de boas-vindas e a exposição do nosso cardápio, com todos os pratos disponíveis para pronta entrega. No Home também informaremos alguns dados essenciais da empresa, como nosso contato e nossa localização.
Na página “Faça seu pedido”, criaremos campos para que o cliente informe seus dados (nome, telefone, endereço, número da casa, etc.). Também haverá opções relacionadas ao pedido, como a escolha do item, a quantidade e um campo para observações, caso o cliente queira solicitar alguma alteração no prato selecionado.
Teremos ainda uma página dedicada a descrever a nossa empresa, explicando nossa origem, nossa área de atuação e os valores que defendemos. Além disso, haverá uma página exclusiva detalhando todos os serviços que oferecemos à nossa clientela.
