# projetoBarbearia

Nosso projeto consiste na criação de um site para serviços de barbearia, oferecendo funcionalidades como agendamento e catálogo de cortes. O objetivo é automatizar a gestão de uma barbearia fictícia, melhorando a organização de horários e serviços oferecidos.
